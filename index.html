_babe provides a basic architecture for browser-based experiments. It offers a simple design template with great convenience functions and many fully worked out examples. You can quickly create behavioral experiments which run inside any modern browser, so that experiments can easily be shared with other researchers to promote your work and support open science.

To collect the data from your experiments, _babe provides a server app with functionality for storing and retrieving the data from your experiments in a database. Thereby _babe allows for multiple methods of deployment without having to change the experiment itself. _babe makes it easy to use crowdsourcing platforms like Amazon's MechanicalTurk or Prolific to recruit participants.

Read the overview section to learn more about _babe in general. Follow the quick start quide to get you up an running with your own experiment. If you need more information, there is also extensive documentation. Or, you could have a look at some templates and examples:

  • Departure Point: minimal template to start developing your own experiment
  • Showroom: display of some of _babe's functionality
